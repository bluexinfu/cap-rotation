## [0.2.0] - 2026-06-16

落實 [圖表系統審查報告](docs/archive/圖表系統審查報告.md) 的全部發現，並新增相對強度座標尺度。

### Added
- **座標尺度切換：金額 ↔ 相對強度**（F-V14）：右上新增 `normSel` 切換。相對強度模式以各板塊**自身日淨買超標準差 σ** 標準化（不減均值、僅除以 σ，故象限歸屬與絕對模式一致），消除「大型股永遠最大」的偏誤，讓「輪動」可跨板塊比較。資訊卡加顯「相對強度 σ」。設計理由見 [ADR-0003](docs/decisions/ADR-0003-relative-strength-normalization.md)。（原 ROADMAP R3 — 審查報告列為「最該補的一刀」）

### Fixed
- 🔴 **時間基準不同步**：修正時間軸停在兩交易日之間時，泡泡位置（內插 `S.pos`）與資訊卡（整數交易日 `S.cur`）講不同一天，導致象限/顏色與文字打架。採「非播放時 `setPos` 以 `Math.round` 吸附整數交易日」（F-V3, F-V5；見 [ADR-0002](docs/decisions/ADR-0002-time-base-alignment.md)）。

### Changed
- 資訊卡象限文字加上方位，例：「退潮（左下）」，由 `QORI` 提供，抗誤讀（F-V5；原 R2）。
- 縱軸正名：「資金加速度」→「資金動能（力道變化·億/天）」，避免誤解為物理二階導數（F-V1；原 R4）。
- 圖例註明雙週期：泡泡顏色＝近5日方向、大小＝近20日量級（F-V2；原 R5）。

### Docs
- 建立完整專案治理體系：產品定位、PRD、變更治理、流程與範本、ROADMAP、決策紀錄（ADR-0001/0002/0003）。
- 將《圖表系統審查報告》歸檔至 `docs/archive/`（發現已全數處理）；專案根目錄舊版 HTML 移入 `archive/`。

## [0.1.0] - 2026-06-16

首個有文件紀錄的基準版本（審查報告修正前的狀態）。

### Added
- **RRG 四象限輪動泡泡圖**（F-V1）：X＝近5日淨買超水位、Y＝資金動能、雙 log 軸，四象限（漲潮/輪動/觀望/退潮）。
- **泡泡編碼**（F-V2）：顏色反映近5日方向，大小反映近20日累計。
- **時間軸播放／拖曳 + 輪動軌跡**（F-V3）。
- **四象限統計卡，可點選篩選**（F-V4）。
- **泡泡資訊卡**（F-V5）：近5日淨買超、動能、象限。
- **龍頭股 K 線面板**（F-V6）：個股 OHLC／指數走勢線雙模式。
- **時間範圍選擇、板塊搜尋/篩選、顯示設定、資金熱力圖、說明彈窗、當日摘要列**（F-V7~V13）。
- **資料管線**（F-D1~D3）：`build_data.py` 從證交所 T86 + MI_INDEX 計算各產業資金流、類股指數、龍頭股 OHLC，純標準庫、零相依。
- **每交易日自動更新**（F-D4）：GitHub Actions 台灣時間約 18:30 自動跑、有變動才 commit。
- **離線備援**（F-V12）：`index.html` 內嵌資料，雙擊即可離線使用。

> 此版本已知的介面矛盾與方法論限制，已於 v0.2.0 全數處理。
