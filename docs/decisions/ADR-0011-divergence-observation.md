# ADR-0011：量價背離「觀察標記」的界線（R9）

- 狀態：已採納
- 日期：2026-07-06
- 關聯：R9（ROADMAP 長期項）、ADR-0005（EMA 平滑）、ADR-0008（σ 中性門檻）、[非目標 §6](../00-product-brief.md)

## 背景

R9 一直放在「長期/待評估」，卡的不是技術，是紅線：量價背離標記很容易滑向「買賣訊號」。
本 ADR 的目的就是把「觀察」與「訊號」的界線寫死，讓功能能做、而且以後不會被自己越改越像投顧。

「資金流」與「類股指數」本來就同時畫在系統裡（左圖＋右側 K 線），使用者已經可以人眼對照；
本功能只是把「兩條線不同步」這個**可驗證的事實**自動標出來，省去逐板塊掃描。

## 決策

新增「量價背離標記」，定義兩種**不同步狀態**（v1 從嚴，寧漏勿濫）：

1. **資金進、價未跟**：近 5 日資金流相對強度 ≥ +1σ，且類股指數近 5 日報酬 ≤ 0。
2. **價漲、資金退**：類股指數近 5 日報酬 ≥ +1 個自身「5日報酬σ」，且資金流相對強度 ≤ −1σ。

呈現與範圍：

- 泡泡標記 `⇄`（中性色，不用紅綠金），tooltip 陳述**兩個事實**（例：「近5日法人日均 +12億（+1.3σ）；類股指數同期 −1.8%」）與一句提醒（成因多種：吸籌/出貨/產業事件/雜訊，需自行研究）。
- 僅「證交所產業」維度（AI 主題無類股指數；以成員股價替代的方案留待 v2 評估）。
- **預設關閉**，設定裡開啟——與逆勢強勢同級的進階觀察。
- 門檻沿用全站 σ 口徑（ADR-0008 的 1σ），**不另設更敏感門檻**去「獵訊號」。
- 資金端隨「法人別」檢視切換（A案）：看外資時標的是「外資進、價未跟」。

## 紅線判準（本 ADR 的核心）

**訊號燈＝替使用者做多空解讀；觀察＝陳述可驗證的事實。**

本功能只說「資金與價格這兩條線目前不同步」，不說這代表什麼、更不說該做什麼。
具體護欄：

- 不排序、不排名、不做「背離強度分數」。
- 不用多空色彩（不紅不綠不金），標記本身不隱含方向判斷。
- tooltip 文案只有數字事實＋「成因多種」提醒，禁用「吸籌」「出貨」「買點」等結論性詞彙。
- 預設關閉：使用者要主動選擇才看得到。
- 誠實揭露：法人資料 T+1 落後，「價未跟」可能隔天就跟上；5 日窗雜訊已知（寫進 ⓘ）。

**預設拒絕的未來需求**（除非新 ADR 推翻非目標）：
「背離＋漲潮＝提示」之類的組合條件、背離推播通知、背離排行榜、把 ⇄ 上色成多空。

## 替代方案（已否決）

- **20 日窗背離**：更穩但太遲鈍，與本產品「早點看到苗頭」定位不合；且 5 日窗與全站 X 軸口徑一致。
- **價格用龍頭股而非類股指數**：龍頭股個股事件雜訊大；類股指數才代表「板塊的價」。
- **AI 主題用成員平均報酬**：成員僅 3 檔、易被單一個股綁架，v1 不做。

## 後果

- 正面：把「量價對照」從人眼掃描變成自動標記，強化 UC5（減碼預警）與研究效率。
- 負面/風險：即使文案中性，使用者仍可能把 ⇄ 當訊號──以預設關閉＋ⓘ 說明緩解；
  兩個 1σ 門檻的組合條件出現頻率低，可能整週無標記（這是「從嚴」的刻意結果，不是 bug）。
